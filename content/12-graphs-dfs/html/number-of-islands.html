Signal: componente conectada em grid
Tempo: O(R * C)
Espaco: O(R * C) no pior caso
Erro comum: esquecer de marcar visitado cedo

Prompt

Conte quantas regioes de terra conectadas por cima, baixo, esquerda e direita existem em uma grade de agua e terra.

Hints

  1. Uma ilha inteira deve ser consumida assim que a primeira celula dela for encontrada.
  2. Marcar in-place simplifica a explicacao.
  3. A pergunta real e: quantas DFS completas eu preciso iniciar?

Naive Path

Relancar exploracoes sem uma nocao global de visitado reprocessa a mesma massa de terra varias vezes.

Optimized Path

Varra a grade. Quando achar uma nova terra, incremente a resposta e rode DFS para zerar toda a ilha atual.

Interview Note

O melhor discurso aqui conecta tres ideias: grid como grafo, ilha como componente e marca de visitado como bloqueio de recontagem.

Final Solution

def numIslands(grid: list[list[str]]) -> int:
    if not grid or not grid[0]:
        return 0

    rows = len(grid)
    cols = len(grid[0])
    islands = 0

    def dfs(row: int, col: int) -> None:
        if row < 0 or row >= rows or col < 0 or col >= cols:
            return
        if grid[row][col] != "1":
            return

        grid[row][col] = "0"
        dfs(row + 1, col)
        dfs(row - 1, col)
        dfs(row, col + 1)
        dfs(row, col - 1)

    for row in range(rows):
        for col in range(cols):
            if grid[row][col] == "1":
                islands += 1
                dfs(row, col)

    return islands
