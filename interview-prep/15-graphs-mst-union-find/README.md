## Como estudar este topico

1. Leia [01-conceitos](./01-conceitos.md) para montar o modelo mental de MST, Kruskal, Prim e Union-Find.
2. Resolva os problemas em markdown explicando em voz alta por que uma aresta pode ser aceita ou rejeitada sem criar ciclo.
3. Abra os HTMLs localmente para revisar prompt, hints, caminho ingenuo, caminho otimizado e analise de complexidade.
