Two Sum

Transforme “achar dois números” em “para este número, o complemento já apareceu?”. Guardar índices evita reutilizar o mesmo elemento.

Controles da simulação

Raciocínio atual

ingênuaO(n²) tempo · O(1) espaço

Solução ingênua

Teste todos os pares i, j com j > i até encontrar a soma igual ao alvo.

  • Tempo: O(n²), todos os pares possíveis.
  • Espaço: O(1).

Solução otimizada

Percorra uma vez. Para cada valor, calcule target - valor. Se o complemento já está no map, retorne o índice guardado e o atual.

Ordem importa: verificar antes de inserir impede usar o mesmo elemento duas vezes.

Big O

Tempo otimizado: O(n) com lookup/inserção amortizados O(1).

Espaço otimizado: O(n), pois o map pode guardar quase todos os valores.

Como falar em entrevista

  • Mostre primeiro a força bruta para estabelecer correção.
  • Explique a troca memória por tempo.
  • Diga que o map guarda valor → índice.
  • Reforce a garantia de uma solução válida, se aplicável.

Código de exemplo

function twoSum(nums, target) {
  const seen = new Map();

  for (let i = 0; i < nums.length; i += 1) {
    const complement = target - nums[i];
    if (seen.has(complement)) return [seen.get(complement), i];
    seen.set(nums[i], i);
  }

  return [];
}
function twoSum(nums: number[], target: number): number[] {
  const seen = new Map<number, number>();

  for (let i = 0; i < nums.length; i += 1) {
    const complement = target - nums[i];
    if (seen.has(complement)) return [seen.get(complement)!, i];
    seen.set(nums[i], i);
  }

  return [];
}
