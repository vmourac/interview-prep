Parallel Courses

Cada camada de nós com indegree 0 representa um semestre inteiro que pode ser feito em paralelo.

Controles da simulação

Raciocínio atual

Foco da entrevista: A fila atual, antes de processar a camada, é exatamente o conjunto de cursos daquele semestre.
ingênuaexponencial tempo · O(V + E) espaço
otimizadaO(V + E) tempo · O(V + E) espaço

Solução ingênua

Testar agrupamentos de cursos por tentativa e erro cria muitas combinações equivalentes.

Solução otimizada

Rode Kahn por níveis: processe o tamanho atual da fila e conte um semestre por camada.

Big O

A organização por camadas não muda o custo linear de Kahn.

Como falar em entrevista

  • Diga que cursos liberados juntos podem ser feitos juntos.
  • Conte completed para detectar ciclo.
  • Retorne -1 quando sobram cursos não processados.

Código de referência

function minimumSemesters(n, relations) {
  const graph = Array.from({ length: n + 1 }, () => []);
  const indegree = Array(n + 1).fill(0);
  for (const [prev, next] of relations) {
    graph[prev].push(next);
    indegree[next] += 1;
  }
  const queue = [];
  for (let course = 1; course <= n; course += 1) {
    if (indegree[course] === 0) queue.push(course);
  }
  let semesters = 0, completed = 0, head = 0;
  while (head < queue.length) {
    const levelSize = queue.length - head;
    semesters += 1;
    for (let i = 0; i < levelSize; i += 1) {
      const course = queue[head++];
      completed += 1;
      for (const next of graph[course]) {
        indegree[next] -= 1;
        if (indegree[next] === 0) queue.push(next);
      }
    }
  }
  return completed === n ? semesters : -1;
}
