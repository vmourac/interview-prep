Valid Palindrome

Compare caracteres relevantes das pontas para o centro, ignorando pontuação e caixa sem criar uma cópia completa da string.

Solução ingênua

Filtre caracteres alfanuméricos, converta para minúsculas e compare a string limpa com sua reversa.

  • Tempo: O(n).
  • Espaço: O(n), pela string filtrada.

Solução otimizada

Mantenha left e right. Pule símbolos dos dois lados e compare apenas caracteres válidos normalizados.

Invariante: tudo fora dos ponteiros já foi validado ou ignorado por não ser alfanumérico.

Big O

Tempo otimizado: O(n), cada ponteiro atravessa a string no máximo uma vez.

Espaço otimizado: O(1), só dois índices e comparações temporárias.

Como falar em entrevista

  • Repita a regra: só alfanuméricos e case-insensitive.
  • Explique por que pular caracteres não altera a resposta.
  • Teste exemplos como “A man, a plan, a canal: Panama”.
  • Mostre que a otimização evita cópia da string.

Código de exemplo

function isPalindrome(s) {
  let left = 0;
  let right = s.length - 1;
  const isAlphaNum = (ch) => /^[a-z0-9]$/i.test(ch);

  while (left < right) {
    while (left < right && !isAlphaNum(s[left])) left += 1;
    while (left < right && !isAlphaNum(s[right])) right -= 1;
    if (s[left].toLowerCase() !== s[right].toLowerCase()) return false;
    left += 1;
    right -= 1;
  }

  return true;
}
