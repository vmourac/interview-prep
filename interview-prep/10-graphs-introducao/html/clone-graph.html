Transit Map Drill

Clone Graph

Este prompt mede se voce sabe copiar uma rede com ciclos preservando a identidade estrutural de cada no por meio de um mapa `original -> clone`.

Nota de fonte: este material e sintetizado a partir do topico do curso, porque a aula original nao tem captions nem transcript disponiveis.

Traversal: O(V + E)
Aux Space: O(V)
Output Space: O(V + E)
Core Idea: one clone per original node

Prompt

Dado um no de entrada em um grafo nao direcionado e conexo, devolva uma copia profunda de toda a estrutura.

Why Interviewers Ask It

O objetivo e ver se voce reconhece que ciclos quebram uma copia recursiva ingenua e que o mapa de clones e obrigatorio.

Naive Route

Tentar clonar cada vizinho recursivamente sem memoizacao. Isso duplica vertices ou entra em recursao infinita quando o grafo volta para um no ja visto.

Optimized Route

Use DFS com um dicionario que associa cada no original ao seu clone. Quando o no reaparece, reutilize a copia ja pronta.

Reasoning Steps

  1. Trate `None` logo no inicio.
  2. Ao ver um no novo, crie sua copia e salve no mapa.
  3. Clone os vizinhos recursivamente.
  4. Se um vizinho ja foi visto, conecte a copia existente.
  5. Retorne o clone do no de entrada.

Final Solution

class Node:
    def __init__(self, val=0, neighbors=None):
        self.val = val
        self.neighbors = neighbors if neighbors is not None else []


class Solution:
    def cloneGraph(self, node: "Node | None") -> "Node | None":
        if node is None:
            return None

        clones = {}

        def dfs(current: "Node") -> "Node":
            if current in clones:
                return clones[current]

            copy = Node(current.val)
            clones[current] = copy

            for neighbor in current.neighbors:
                copy.neighbors.append(dfs(neighbor))

            return copy

        return dfs(node)
