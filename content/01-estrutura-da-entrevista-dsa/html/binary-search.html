Estudo orientado por entrevista

Binary
Search

Encontre o indice de um valor em um array ordenado. Se ele nao existir, retorne -1.

Hints

  • A ordenacao e a pista principal do problema.
  • Uma comparacao no meio elimina metade do intervalo.
  • Mantenha o invariante de que o alvo, se existir, esta entre esquerda e direita.

Caminho ingenuo

Varra o array linearmente. Serve como baseline, mas ignora a informacao mais importante do enunciado: a entrada esta ordenada.

Caminho otimizado

Compare o elemento do meio com o target e descarte metade do espaco de busca a cada passo.

Complexidade

Ingenua: tempo O(n)
Ingenua: espaco O(1)
Otimizada: tempo O(log n)
Otimizada: espaco O(1)

O salto para O(log n) vem da divisao repetida do intervalo pela metade. A cada iteracao, varias comparacoes futuras deixam de existir.

Solucao final

def binary_search(nums, target):
    esquerda = 0
    direita = len(nums) - 1

    while esquerda <= direita:
        meio = (esquerda + direita) // 2

        if nums[meio] == target:
            return meio
        if nums[meio] < target:
            esquerda = meio + 1
        else:
            direita = meio - 1

    return -1
