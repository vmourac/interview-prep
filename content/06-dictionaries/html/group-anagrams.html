Group Anagrams

O entrevistador quer ver se voce percebe que anagramas nao precisam ser comparados dois a dois. O salto real e inventar uma chave canonica que transforme equivalencia de palavras em simples agrupamento de hash map.

hash map canonical key strings
Naive
O(n^2 * k log k)
Optimized
O(n * k log k)
Memory
O(n * k)

Prompt

Dada uma lista de strings, agrupe palavras que sao anagramas entre si. A ordem dos grupos e a ordem interna podem variar.

Hints

  1. Como decidir rapidamente se duas palavras pertencem ao mesmo grupo?
  2. Existe uma representacao unica para todas as palavras que usam as mesmas letras?
  3. Se essa representacao existir, qual estrutura natural guarda todas as palavras associadas a ela?

Naive Path

Compare palavras em pares e, a cada comparacao, cheque equivalencia ordenando caracteres ou contando frequencias. Funciona, mas repete trabalho demais.

Optimized Path

Converta cada palavra em uma assinatura canonica, como seus caracteres ordenados. Use essa assinatura como chave do dicionario e acumule ali todas as palavras equivalentes.

Complexity

  • Tempo: O(n * k log k), dominado pela ordenacao de cada palavra.
  • Espaco: O(n * k), para armazenar chaves e grupos.
  • Entrevista: destaque que o ganho vem de trocar comparacao entre pares por classificacao via chave.

Final Solution

from collections import defaultdict


def groupAnagrams(strs: list[str]) -> list[list[str]]:
    groups = defaultdict(list)

    for word in strs:
        key = "".join(sorted(word))
        groups[key].append(word)

    return list(groups.values())
