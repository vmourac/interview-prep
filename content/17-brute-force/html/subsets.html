Subsets

O discurso certo aqui e mostrar que cada elemento abre uma bifurcacao binaria: eu incluo ou ignoro, e a arvore inteira representa o espaco de respostas.

[]
[1]
[2]
[1,2]
Signal: gerar todas as configuracoes
Tempo: O(n * 2^n)
Espaco: O(n * 2^n)
Chave: incluir ou nao incluir

Prompt

Dado um array sem repeticao, retorne todos os subconjuntos possiveis em qualquer ordem.

Por que cai em entrevista

  1. Explicita a arvore binaria de escolhas.
  2. Treina copia de estado no ponto certo.
  3. Forca a justificar por que o custo exponencial e inevitavel.

Naive Path

Gerar tamanhos separadamente ou iterar mascaras de bits funciona, mas comunica menos sobre o modelo geral de backtracking.

Optimized Path

Use DFS por indice. Em cada elemento, siga um ramo sem usar o valor e outro ramo incluindo o valor.

Interview Note

O entrevistador quer ouvir que existem `2^n` respostas possiveis e que copiar o caminho em cada folha explica o fator `n` adicional.

Final Solution

def subsets(nums: list[int]) -> list[list[int]]:
    result = []
    path = []

    def dfs(index: int) -> None:
        if index == len(nums):
            result.append(path[:])
            return

        dfs(index + 1)

        path.append(nums[index])
        dfs(index + 1)
        path.pop()

    dfs(0)
    return result
