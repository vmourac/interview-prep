Merge Two Sorted Lists

A entrada já está ordenada. A entrevista quer ver se você aproveita esse fato para fazer um merge linear, sem ordenar tudo de novo.

padrão: dummy node tail aponta para o fim da resposta O(n + m) tempo

Controles da simulação

Raciocínio atual

Por que dummy ajuda? Ele evita o caso especial “a resposta ainda não tem cabeça”. O retorno final é sempre dummy.next.
ingênuaO(k log k) tempo · O(k) espaço
otimizadaO(n + m) tempo · O(1) espaço

Solução ingênua

Copie todos os valores para um array, ordene e reconstrua a lista. Funciona, mas ignora a propriedade mais importante: as duas listas já vêm ordenadas.

  • Tempo: com k = n + m, ordenar custa O(k log k).
  • Espaço: array e nova lista custam O(k).

Solução otimizada

Compare apenas as cabeças ainda não consumidas. O menor nó é anexado ao fim da resposta, e só a lista que forneceu esse nó avança.

Invariante: dummy.next até tail já está ordenado, e tail é o último nó da resposta parcial.

Big O

Tempo otimizado: O(n + m), porque cada nó é anexado uma vez.

Espaço otimizado: O(1) extra se reaproveitarmos os nós existentes. O dummy é constante.

Como falar em entrevista

  • Comece explicitando que as listas já estão ordenadas.
  • Defina o papel de dummy e tail.
  • Explique que, quando uma lista acaba, o restante da outra já está ordenado.
  • Mostre que não há nested loop: cada avanço consome um nó de alguma lista.

Código de referência

function mergeTwoLists(list1, list2) {
  const dummy = { value: 0, next: null };
  let tail = dummy;

  while (list1 !== null && list2 !== null) {
    if (list1.value <= list2.value) {
      tail.next = list1;
      list1 = list1.next;
    } else {
      tail.next = list2;
      list2 = list2.next;
    }
    tail = tail.next;
  }

  tail.next = list1 !== null ? list1 : list2;
  return dummy.next;
}
type ListNode = {
  value: number;
  next: ListNode | null;
};

function mergeTwoLists(
  list1: ListNode | null,
  list2: ListNode | null
): ListNode | null {
  const dummy: ListNode = { value: 0, next: null };
  let tail: ListNode = dummy;

  while (list1 !== null && list2 !== null) {
    if (list1.value <= list2.value) {
      tail.next = list1;
      list1 = list1.next;
    } else {
      tail.next = list2;
      list2 = list2.next;
    }
    tail = tail.next;
  }

  tail.next = list1 !== null ? list1 : list2;
  return dummy.next;
}
