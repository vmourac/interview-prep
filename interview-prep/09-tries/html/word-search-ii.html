Build trie: O(S)
Board search: pior caso O(M * N * 4 * 3^(L - 1))
Trie space: O(S)
DFS stack: O(L)

Prompt

Dado um board de letras e uma lista de palavras, retorne todas as palavras que podem ser montadas por adjacencia horizontal ou vertical sem reutilizar a mesma celula no mesmo caminho.

Hints

  1. Buscar cada palavra separadamente repete muito trabalho.
  2. Se o caminho atual nao e prefixo de nenhuma palavra, a DFS deve morrer na hora.
  3. Deduplicacao de resposta pode ser feita no proprio no terminal.

Naive Path

Rodar backtracking completo para cada palavra da lista. O board inteiro pode ser revisitato muitas vezes para prefixos semelhantes.

Optimized Path

Insira todas as palavras em uma trie e faca uma DFS unica por celula inicial. O caminho no board avanca junto com o caminho na trie; quando o prefixo some, a busca para.

Interview Note

O melhor discurso aqui nao vende um milagre de Big O. Ele explica que o pior caso ainda e alto, mas o pruning por prefixo derruba brutalmente o trabalho repetido na pratica.

Final Solution

class TrieNode:
    def __init__(self):
        self.children = {}
        self.word = None


def findWords(board: list[list[str]], words: list[str]) -> list[str]:
    root = TrieNode()

    for word in words:
        node = root
        for char in word:
            if char not in node.children:
                node.children[char] = TrieNode()
            node = node.children[char]
        node.word = word

    rows = len(board)
    cols = len(board[0]) if board else 0
    found = []

    def dfs(row: int, col: int, node: TrieNode) -> None:
        char = board[row][col]

        if char not in node.children:
            return

        next_node = node.children[char]

        if next_node.word is not None:
            found.append(next_node.word)
            next_node.word = None

        board[row][col] = "#"

        for delta_row, delta_col in ((1, 0), (-1, 0), (0, 1), (0, -1)):
            new_row = row + delta_row
            new_col = col + delta_col

            if 0 <= new_row < rows and 0 <= new_col < cols and board[new_row][new_col] != "#":
                dfs(new_row, new_col, next_node)

        board[row][col] = char

    for row in range(rows):
        for col in range(cols):
            dfs(row, col, root)

    return found
