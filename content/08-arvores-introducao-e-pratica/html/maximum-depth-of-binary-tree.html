Maximum Depth of Binary Tree

Profundidade máxima pode ser calculada com DFS iterativa: cada item da pilha carrega um nó e sua profundidade.

Raciocínio atual

Foco de entrevista: O caso base null → 0 evita off-by-one e deixa folha retornar 1.
ingênuaO(n) tempo · O(n) espaço

Solução ingênua

Faça BFS por níveis com fila e conte quantas camadas existem. É claro, mas pode guardar um nível inteiro.

  • Cada nó entra e sai da fila uma vez.
  • A fila pode conter O(n) nós em árvore larga.

Solução otimizada

Use DFS iterativa e empilhe cada filho com a profundidade do pai mais 1.

Invariante: cada par na pilha contém a profundidade correta do nó recebido.

Big O

Tempo O(n) porque todos os nós são visitados; espaço O(h) pela pilha explícita, onde h é a altura.

Como falar em entrevista

  • Defina se profundidade conta nós ou arestas; aqui conta nós.
  • Explique por que folha retorna 1.
  • Compare BFS e DFS sem fingir que uma visita menos nós.

Código de referência

function maxDepth(root) {
  if (root === null) return 0;
  let deepest = 0;
  const stack = [[root, 1]];
  while (stack.length) {
    const [node, depth] = stack.pop();
    deepest = Math.max(deepest, depth);
    if (node.left !== null) stack.push([node.left, depth + 1]);
    if (node.right !== null) stack.push([node.right, depth + 1]);
  }
  return deepest;
}
type TreeNode = { val: number; left: TreeNode | null; right: TreeNode | null };

function maxDepth(root: TreeNode | null): number {
  if (root === null) return 0;
  let deepest = 0;
  const stack: Array<[TreeNode, number]> = [[root, 1]];
  while (stack.length) {
    const [node, depth] = stack.pop()!;
    deepest = Math.max(deepest, depth);
    if (node.left !== null) stack.push([node.left, depth + 1]);
    if (node.right !== null) stack.push([node.right, depth + 1]);
  }
  return deepest;
}
