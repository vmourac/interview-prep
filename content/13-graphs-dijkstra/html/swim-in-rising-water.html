Flood Navigation

Swim in Rising Water

O tempo minimo equivale ao menor gargalo possivel no caminho. Em outras palavras, queremos o trajeto cujo maior nivel de agua exigido seja o menor entre todos.

0 2 5 1 7 6 3 4 8
Signal: minimize o maior nivel requerido
State: (time, row, col)
Relax: max(current, next cell)
Tempo: O(R * C * log(R * C))

Prompt

Voce so entra em uma celula quando a agua ja subiu o bastante para cobrir sua elevacao. Descubra o menor tempo que torna possivel sair da origem e chegar ao destino.

Hints

  1. O custo do caminho e o maximo das elevacoes visitadas.
  2. O custo inicial ja e a elevacao da origem.
  3. A melhor proxima celula e a que exige o menor tempo acumulado.

Naive Path

Subir o tempo aos poucos e repetir testes de conectividade recalcula demais a mesma grade para tempos diferentes.

Optimized Path

Use Dijkstra com custo minimax. Ao mover, o novo tempo necessario vira o maximo entre o tempo atual e a elevacao da proxima celula.

Interview Note

Este problema e praticamente irmao de Path With Minimum Effort. A mudanca central esta apenas no significado do custo local.

Final Solution

import heapq


def swimInWater(grid: list[list[int]]) -> int:
    n = len(grid)
    inf = float("inf")
    best = [[inf] * n for _ in range(n)]
    best[0][0] = grid[0][0]
    heap = [(grid[0][0], 0, 0)]
    directions = ((1, 0), (-1, 0), (0, 1), (0, -1))

    while heap:
        current_time, row, col = heapq.heappop(heap)

        if row == n - 1 and col == n - 1:
            return current_time

        if current_time > best[row][col]:
            continue

        for delta_row, delta_col in directions:
            next_row = row + delta_row
            next_col = col + delta_col

            if not (0 <= next_row < n and 0 <= next_col < n):
                continue

            next_time = max(current_time, grid[next_row][next_col])

            if next_time < best[next_row][next_col]:
                best[next_row][next_col] = next_time
                heapq.heappush(heap, (next_time, next_row, next_col))

    return -1
