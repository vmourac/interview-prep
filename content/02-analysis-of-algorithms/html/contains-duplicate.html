Contains Duplicate

A pergunta operacional é “já vi esse valor antes?”. Um set troca memória por uma resposta linear.

Solução ingênua

Compare cada elemento com todos os elementos à direita. Assim que encontrar igualdade, retorne true.

  • Tempo: O(n²).
  • Espaço: O(1).

Solução otimizada

Percorra o array uma vez. Se o valor atual já está no set, encontramos duplicata; caso contrário, adicione-o.

Invariante: antes de processar nums[i], o set contém exatamente os valores à esquerda de i.

Big O

Tempo otimizado: O(n), assumindo operações de set O(1) amortizadas.

Espaço otimizado: O(n), se não houver repetição até o final.

Como falar em entrevista

  • Declare a força bruta para mostrar o baseline.
  • Diga que a otimização responde “já vi antes?”.
  • Explique a troca clássica: mais memória, menos tempo.
  • Mencione retorno antecipado ao achar a primeira duplicata.

Código de exemplo

function containsDuplicate(nums) {
  const seen = new Set();

  for (const value of nums) {
    if (seen.has(value)) return true;
    seen.add(value);
  }

  return false;
}
function containsDuplicate(nums: number[]): boolean {
  const seen = new Set<number>();

  for (const value of nums) {

  }
