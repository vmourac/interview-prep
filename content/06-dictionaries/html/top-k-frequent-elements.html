Prompt

Dado um array de inteiros e um valor `k`, retorne os `k` elementos mais frequentes em qualquer ordem.

Hints

  1. Qual estrutura resolve a contagem rapidamente?
  2. Voce realmente precisa ordenar todos os valores distintos?
  3. Qual e a maior frequencia possivel de um numero em um array de tamanho `n`?

Naive Path

Conte com hash map, transforme em lista de pares e ordene tudo por frequencia decrescente. A solucao e correta, mas paga ordenacao global mesmo quando so precisamos dos primeiros `k`.

Optimized Path

Monte baldes por frequencia. Como uma frequencia nunca passa de `n`, podemos indexar uma lista de baldes e depois ler do maior para o menor ate preencher a resposta.

Complexity

  • Tempo: O(n), somando contagem, distribuicao em baldes e coleta final.
  • Espaco: O(n), pelo mapa e pelos baldes.
  • Entrevista: a resposta forte explica por que frequencia pertence a uma faixa limitada.

Final Solution

def topKFrequent(nums: list[int], k: int) -> list[int]:
    freq = {}
    for value in nums:
        freq[value] = freq.get(value, 0) + 1

    buckets = [[] for _ in range(len(nums) + 1)]
    for value, count in freq.items():
        buckets[count].append(value)

    result = []
    for count in range(len(buckets) - 1, 0, -1):
        for value in buckets[count]:
            result.append(value)
            if len(result) == k:
                return result

    return result
