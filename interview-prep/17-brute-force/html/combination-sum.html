Chave: ordem e pruning

Prompt

Dado um conjunto de candidatos positivos distintos e um alvo, retorne todas as combinacoes cuja soma seja exatamente esse alvo.

Hints

  1. Reusar numero significa ficar no mesmo indice apos escolher.
  2. Evite permutacoes duplicadas impondo ordem nao decrescente.
  3. Com array ordenado, `value > remaining` encerra o loop.

Naive Path

Tentar qualquer candidato em qualquer ordem gera varias copias estruturais da mesma combinacao e deixa deduplicacao para depois.

Optimized Path

Ordene os candidatos, carregue `remaining` na recursao e so avance a partir do indice atual para evitar permutacoes repetidas.

Interview Note

O ponto forte nao e inventar uma formula fechada, e sim mostrar que pruning e seguro porque todos os numeros sao positivos e o array esta ordenado.

Final Solution

def combinationSum(candidates: list[int], target: int) -> list[list[int]]:
    candidates.sort()
    result = []
    path = []

    def dfs(start: int, remaining: int) -> None:
        if remaining == 0:
            result.append(path[:])
            return

        for index in range(start, len(candidates)):
            value = candidates[index]

            if value > remaining:
                break

            path.append(value)
            dfs(index, remaining - value)
            path.pop()

    dfs(0, target)
    return result
