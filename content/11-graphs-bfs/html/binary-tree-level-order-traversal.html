Metro Layer Walk

Binary Tree Level Order Traversal

Este e o exercicio em que BFS aparece no seu formato mais limpo. O objetivo nao e achar um caminho, mas materializar a arvore nivel por nivel usando a fila como trilho principal.

Nivel 0: raiz
Nivel 1: esquerda
Nivel 1: direita
Nivel 2: filhos
Fila prepara proximo nivel
Signal: level by level
Tempo: O(N)
Fila auxiliar: O(W)
Saida: O(N)

Prompt

Dada a raiz de uma arvore binaria, retorne uma lista de listas com os valores agrupados por nivel, da esquerda para a direita.

Por que cai em entrevista

  1. Mostra se voce associa fila com exploracao por camadas.
  2. Forca a separar nivel atual de proximo nivel sem confundir a fila dinamica.
  3. Serve de base para zigzag, right side view e media por nivel.

Naive Path

Fazer DFS com parametro de profundidade tambem resolve, mas a intuicao fica menos direta para um problema explicitamente orientado a niveis.

Optimized Path

Rode BFS. No inicio de cada rodada, congele `len(queue)`. Esse tamanho representa exatamente o nivel atual; tudo o que for enfileirado depois pertence ao proximo.

Interview Note

A frase que mais agrega aqui e: eu congelei o tamanho da fila antes do loop interno, entao consigo consumir um nivel inteiro sem misturar filhos recem-descobertos.

Final Solution

from collections import deque


def levelOrder(root):
    if root is None:
        return []

    result = []
    queue = deque([root])

    while queue:
        level_size = len(queue)
        current_level = []

        for _ in range(level_size):
            node = queue.popleft()
            current_level.append(node.val)

            if node.left is not None:
                queue.append(node.left)
            if node.right is not None:
                queue.append(node.right)

        result.append(current_level)

    return result
