Topico 03 - Treino guiado

Merge Two Sorted Lists

Sintese de estudo em formato de entrevista. O foco aqui e explorar o padrao de dummy node e justificar por que o merge correto e linear no total de nos.

Dummy node
Merge linear
O(n + m)

Prompt

Receba duas listas ligadas simples ordenadas em ordem crescente e devolva uma unica lista ordenada contendo todos os nos.

Hints

  • Pense em um ponteiro que sempre aponta para o final da resposta.
  • Evite tratar a cabeca como um caso especial usando um no sentinela.
  • Quando uma lista acabar, o restante da outra ja esta pronto para anexar.

Caminho ingenuo

Copiar tudo para um array, ordenar e reconstruir resolve o problema, mas adiciona memoria extra e desperdi ca o fato de as entradas ja estarem ordenadas.

Caminho otimizado

Compare as pontas de ambas as listas e sempre ligue o menor no ao final da resposta. Avance apenas a lista que forneceu esse no. O tail caminha uma vez por no anexado.

Complexidade

Tempo: O(n + m)

Espaco: O(1)

Cada comparacao consome um no de alguma lista. Nenhum no volta para processamento futuro.

Solucao final

def mergeTwoLists(l1, l2):
    dummy = ListNode(0)
    tail = dummy

    while l1 is not None and l2 is not None:
        if l1.val <= l2.val:
            tail.next = l1
            l1 = l1.next
        else:
            tail.next = l2
            l2 = l2.next

        tail = tail.next

    if l1 is not None:
        tail.next = l1
    else:
        tail.next = l2

    return dummy.next
