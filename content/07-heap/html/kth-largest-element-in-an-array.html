Kth Largest Element in an Array

A pergunta pede apenas a fronteira dos k maiores, não a ordenação completa. A min-heap de tamanho k mantém visível exatamente o candidato que cairia fora primeiro.

Raciocínio atual

Foco de entrevista: Diga que o topo da min-heap é o menor entre os k maiores vistos até agora; por isso ele vira a resposta final.
ingênuaO(n log n) tempo · O(1)/O(n) espaço
otimizadaO(n log k) tempo · O(k) espaço

Solução ingênua

Ordene o array inteiro em ordem decrescente e leia a posição k - 1. Funciona, mas organiza elementos que nunca serão necessários para a resposta.

  • Tempo: O(n log n) pela ordenação completa.
  • Espaço: depende da implementação do sort; o ponto fraco principal é o tempo global.

Solução otimizada

Percorra os números uma vez. Insira cada valor em uma min-heap; se o tamanho passar de k, remova o menor. Ao final, sobraram os k maiores.

Invariante: a heap nunca tem mais de k elementos e contém os k maiores entre os valores já processados.

Big O

Cada um dos n valores faz operações de heap com altura log k, então o tempo é O(n log k). A heap limitada custa O(k).

Como falar em entrevista

  • Explique que o problema não pede elementos distintos.
  • Mostre por que descartar o menor quando a heap passa de k é seguro.
  • Se perguntarem alternativa, mencione quickselect como tempo médio O(n), mas defenda heap como robusta.

Código de referência

class MinHeap {
  constructor() { this.data = []; }
  peek() { return this.data[0]; }
  push(value) { this.data.push(value); this.bubbleUp(this.data.length - 1); }
  pop() {
    const top = this.data[0];
    const last = this.data.pop();
    if (this.data.length > 0) { this.data[0] = last; this.bubbleDown(0); }
    return top;
  }
  bubbleUp(i) {
    while (i > 0) {
      const p = Math.floor((i - 1) / 2);
      if (this.data[p] <= this.data[i]) break;
      [this.data[p], this.data[i]] = [this.data[i], this.data[p]];
      i = p;
    }
  }
  bubbleDown(i) {
    while (true) {
      let smallest = i;
      for (const child of [2 * i + 1, 2 * i + 2]) {
        if (child < this.data.length && this.data[child] < this.data[smallest]) smallest = child;
      }
      if (smallest === i) break;
      [this.data[i], this.data[smallest]] = [this.data[smallest], this.data[i]];
      i = smallest;
    }
  }
}

function findKthLargest(nums, k) {
  const heap = new MinHeap();
  for (const value of nums) {
    heap.push(value);
    if (heap.data.length > k) heap.pop();
  }
  return heap.peek();
}
class MinHeap {
  data: number[] = [];
  peek(): number { return this.data[0]; }
  push(value: number): void { this.data.push(value); this.bubbleUp(this.data.length - 1); }
  pop(): number {
    const top = this.data[0];
    const last = this.data.pop()!;
    if (this.data.length > 0) { this.data[0] = last; this.bubbleDown(0); }
    return top;
  }
  private bubbleUp(i: number): void {
    while (i > 0) {
      const p = Math.floor((i - 1) / 2);
      if (this.data[p] <= this.data[i]) break;
      [this.data[p], this.data[i]] = [this.data[i], this.data[p]];
      i = p;
    }
  }
  private bubbleDown(i: number): void {
    while (true) {
      let smallest = i;
      for (const child of [2 * i + 1, 2 * i + 2]) {
        if (child < this.data.length && this.data[child] < this.data[smallest]) smallest = child;
      }
      if (smallest === i) break;
      [this.data[i], this.data[smallest]] = [this.data[smallest], this.data[i]];
      i = smallest;
    }
  }
}

function findKthLargest(nums: number[], k: number): number {
  const heap = new MinHeap();
  for (const value of nums) {
    heap.push(value);
    if (heap.data.length > k) heap.pop();
  }
  return heap.peek();
