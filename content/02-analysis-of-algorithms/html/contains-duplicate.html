Contains Duplicate

Pergunta curta, sinal forte. O entrevistador quer ver se voce abandona a comparacao de todos contra todos e passa a tratar "ja vi este valor?" como consulta de estrutura auxiliar.

Tempo Ingenua: O(n^2) | Otimizada: O(n)
Espaco Ingenua: O(1) | Otimizada: O(n)

Prompt

Dado um array de inteiros, retorne true se algum valor aparece pelo menos duas vezes. Caso contrario, retorne false.

array
hash set
big o

Hints

  1. Como voce resolveria sem memoria extra?
  2. Qual pergunta se repete a cada iteracao?
  3. Existe uma estrutura que responde rapidamente se um valor ja apareceu?

Naive Path

  • Fixe um indice inicial.
  • Compare com todos os elementos a frente.
  • Se encontrar igualdade, retorne true.
  • Se examinar todos os pares, retorne false.

Raciocinio correto, mas quadratico.

Optimized Path

  • Mantenha um set com os valores vistos.
  • Ao ler um valor, cheque se ele ja esta no set.
  • Se sim, a duplicata foi provada.
  • Se nao, registre e avance.

A troca principal e memoria extra por um unico percurso linear.

Complexity

  • Tempo: O(n), porque cada elemento e processado uma vez.
  • Espaco: O(n), porque o set pode armazenar todos os elementos.
  • Entrevista: cite explicitamente a troca entre tempo e memoria.

Final Solution

def containsDuplicate(nums: list[int]) -> bool:
    seen = set()

    for value in nums:
        if value in seen:
            return True
        seen.add(value)

    return False
