Solução ingênua

Enumerar subconjuntos de arestas e testar quais formam árvores é combinatório.

Solução otimizada

Prim mantém minDist[i], a conexão mais barata de cada ponto fora da MST para a árvore atual.

Big O

São n iterações; cada uma varre n pontos para escolher e n para atualizar, total O(n²).

Como falar em entrevista

  • Reconheça MST mesmo sem a palavra no enunciado.
  • Explique que o grafo é completo.
  • Defina minDist como melhor aresta para entrar na árvore.

Código de referência

function minCostConnectPoints(points) {
  const n = points.length;
  const inMst = Array(n).fill(false);
  const minDist = Array(n).fill(Infinity);
  minDist[0] = 0;
  let total = 0;

  for (let count = 0; count < n; count += 1) {
    let current = -1;
    for (let i = 0; i < n; i += 1) {
      if (!inMst[i] && (current === -1 || minDist[i] < minDist[current])) current = i;
    }
    inMst[current] = true;
    total += minDist[current];
    const [x1, y1] = points[current];
    for (let next = 0; next < n; next += 1) {
      if (inMst[next]) continue;
      const [x2, y2] = points[next];
      const cost = Math.abs(x1 - x2) + Math.abs(y1 - y2);
      if (cost < minDist[next]) minDist[next] = cost;
    }
  }
  return total;
}
