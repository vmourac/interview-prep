Linked List Cycle

Um ciclo não precisa ser “visto” guardando todos os nós. Dois ponteiros com velocidades diferentes transformam a estrutura circular em encontro inevitável.

padrão: Floyd slow anda 1 fast anda 2 O(1) espaço

Controles da simulação

Raciocínio atual

Intuição: dentro do ciclo, o fast ganha 1 posição relativa por rodada. Em algum momento, a distância modular entre fast e slow vira zero.
ingênuaO(n) tempo · O(n) espaço
otimizadaO(n) tempo · O(1) espaço

Solução ingênua

Guarde cada referência de nó em um Set. Se encontrar um nó já visto, existe ciclo. Se chegar em null, não existe ciclo.

  • Tempo: O(n), cada nó é consultado no máximo uma vez.
  • Espaço: O(n), porque o conjunto pode guardar todos os nós antes de detectar repetição.

Solução otimizada

Use dois ponteiros. slow anda um nó por rodada; fast anda dois. Se não há ciclo, fast encontra null. Se há ciclo, fast encontra slow.

Invariante operacional: só acessamos fast.next.next depois de validar fast !== null e fast.next !== null.

Big O

Tempo otimizado: O(n). Antes do ciclo, fast avança pela parte linear; dentro do ciclo, o encontro ocorre em no máximo o tamanho do ciclo.

Espaço otimizado: O(1), pois só mantemos duas referências.

Como falar em entrevista

  • Comece pela solução com Set para demonstrar clareza.
  • Depois otimize memória com Floyd.
  • Explique a condição segura do loop antes de mover fast dois passos.
  • Justifique o encontro com velocidade relativa, não com “porque sim”.

Código de referência

function hasCycle(head) {
  let slow = head;
  let fast = head;

  while (fast !== null && fast.next !== null) {
    slow = slow.next;
    fast = fast.next.next;

    if (slow === fast) {
      return true;
    }
  }

  return false;
}
type ListNode = {
  value: number;
  next: ListNode | null;
};

function hasCycle(head: ListNode | null): boolean {
  let slow: ListNode | null = head;
  let fast: ListNode | null = head;

  while (fast !== null && fast.next !== null) {
    slow = slow === null ? null : slow.next;
    fast = fast.next.next;

    if (slow === fast) {
      return true;
    }
  }

  return false;
}
