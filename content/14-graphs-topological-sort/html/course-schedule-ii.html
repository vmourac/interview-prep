Signal: DAG de prerequisitos
Tempo: O(V + E)
Espaco: O(V + E)
Ferramenta: indegree + fila

Prompt

Retorne uma ordem valida de cursos. Se o grafo tiver ciclo, a resposta precisa ser lista vazia.

Hints

  1. Defina a aresta como prereq aponta para course.
  2. `indegree == 0` significa curso liberado.
  3. Se faltar curso no final, havia ciclo.

Naive Path

Tentar permutar agendas de cursos por backtracking ignora a estrutura do grafo e explode em combinacoes.

Optimized Path

Use Kahn. Retire cursos liberados, reduza indegree dos dependentes e construa a ordem incrementalmente.

Interview Note

Fale que topological sort responde as duas perguntas ao mesmo tempo: qual ordem funciona e como provar que uma ordem nao existe.

Final Solution

from collections import deque


def findOrder(numCourses: int, prerequisites: list[list[int]]) -> list[int]:
    graph = [[] for _ in range(numCourses)]
    indegree = [0] * numCourses

    for course, prereq in prerequisites:
        graph[prereq].append(course)
        indegree[course] += 1

    queue = deque(course for course in range(numCourses) if indegree[course] == 0)
    order = []

    while queue:
        course = queue.popleft()
        order.append(course)

        for neighbor in graph[course]:
            indegree[neighbor] -= 1
            if indegree[neighbor] == 0:
                queue.append(neighbor)

    return order if len(order) == numCourses else []
