Cost Surface

Min Cost to Connect All Points

Mesmo sem o termo MST no enunciado, o objetivo e construir uma rede que conecte todos os pontos pelo menor custo total possivel.

Signal: conectar tudo com custo minimo
Tempo: O(n^2)
Espaco: O(n)
Estrategia: Prim sem heap

Prompt

Dado um conjunto de pontos no plano, conecte todos eles minimizando a soma das distancias Manhattan usadas.

Hints

  1. O grafo implicito e completo: todo par de pontos tem uma aresta.
  2. Voce nao precisa construir todas as arestas explicitamente para usar Prim.
  3. `min_dist[i]` e o melhor custo atual para puxar o ponto `i` para dentro da MST.

Naive Path

Enumerar arvores candidatas ou comparar muitas combinacoes de conexoes e inviavel porque o espaco de busca explode.

Optimized Path

Cresca a arvore ponto a ponto, sempre anexando o vertice fora da MST com menor custo de conexao conhecido.

Interview Note

O argumento decisivo aqui e escolher Prim porque o custo entre dois pontos pode ser calculado sob demanda, evitando listar e ordenar `O(n^2)` arestas.

Final Solution

def minCostConnectPoints(points: list[list[int]]) -> int:
    n = len(points)
    in_mst = [False] * n
    min_dist = [float("inf")] * n
    min_dist[0] = 0
    total_cost = 0

    for _ in range(n):
        current = -1

        for node in range(n):
            if not in_mst[node] and (current == -1 or min_dist[node] < min_dist[current]):
                current = node

        in_mst[current] = True
        total_cost += min_dist[current]

        x1, y1 = points[current]

        for neighbor in range(n):
            if in_mst[neighbor]:
                continue

            x2, y2 = points[neighbor]
            cost = abs(x1 - x2) + abs(y1 - y2)

            if cost < min_dist[neighbor]:
                min_dist[neighbor] = cost

    return total_cost
