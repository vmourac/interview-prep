Semester Planner

Parallel Courses

A resposta nao e uma lista de vertices. E o numero de ondas de cursos liberados, ou seja, quantas camadas o topological sort leva para consumir o grafo.

Signal: topological sort por niveis
Tempo: O(V + E)
Espaco: O(V + E)
Insight: tamanho atual da fila = um semestre

Prompt

Calcule o numero minimo de semestres para concluir todos os cursos, ou retorne -1 se um ciclo impedir a conclusao.

Hints

  1. Todos os cursos com indegree zero podem ser feitos juntos.
  2. Processe a fila em blocos por nivel.
  3. Se sobrar curso no final, havia ciclo.

Naive Path

Testar diferentes gradeamentos de semestres por tentativa e erro custa caro e nao e necessario quando as dependencias ja determinam as camadas.

Optimized Path

Rode Kahn em BFS por niveis. Cada rodada completa dos nos atualmente liberados corresponde a um semestre.

Interview Note

O pulo do gato e perceber que a fila nao representa uma ordem qualquer, mas exatamente o conjunto de cursos paralelizaveis naquele instante.

Final Solution

from collections import deque


def minimumSemesters(n: int, relations: list[list[int]]) -> int:
    graph = [[] for _ in range(n + 1)]
    indegree = [0] * (n + 1)

    for prev_course, next_course in relations:
        graph[prev_course].append(next_course)
        indegree[next_course] += 1

    queue = deque(course for course in range(1, n + 1) if indegree[course] == 0)
    semesters = 0
    completed = 0

    while queue:
        semesters += 1

        for _ in range(len(queue)):
            course = queue.popleft()
            completed += 1

            for neighbor in graph[course]:
                indegree[neighbor] -= 1
                if indegree[neighbor] == 0:
                    queue.append(neighbor)

    return semesters if completed == n else -1
