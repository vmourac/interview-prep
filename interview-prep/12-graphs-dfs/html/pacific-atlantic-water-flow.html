Prompt

Encontre as celulas que conseguem escoar para o Pacifico e para o Atlantico sob a regra de que a agua so desce ou fica no mesmo nivel.

Hints

  1. Rodar busca por origem repete trabalho demais.
  2. No sentido reverso, da para andar para alturas maiores ou iguais.
  3. Comece simultaneamente das bordas de cada oceano.

Naive Path

Rodar DFS de cada celula para descobrir se algum caminho chega aos dois oceanos gera buscas enormes e redundantes.

Optimized Path

Rode uma DFS reversa das bordas do Pacifico e outra das bordas do Atlantico. A resposta final e a intersecao desses dois conjuntos.

Interview Note

O diferencial nao e decorar a solucao, e sim justificar por que inverter o fluxo derruba o custo de repetir busca para todas as origens.

Final Solution

def pacificAtlantic(heights: list[list[int]]) -> list[list[int]]:
    if not heights or not heights[0]:
        return []

    rows = len(heights)
    cols = len(heights[0])
    pacific_reachable = set()
    atlantic_reachable = set()

    def dfs(row: int, col: int, visited: set[tuple[int, int]]) -> None:
        visited.add((row, col))

        for delta_row, delta_col in ((1, 0), (-1, 0), (0, 1), (0, -1)):
            new_row = row + delta_row
            new_col = col + delta_col

            if new_row < 0 or new_row >= rows or new_col < 0 or new_col >= cols:
                continue
            if (new_row, new_col) in visited:
                continue
            if heights[new_row][new_col] < heights[row][col]:
                continue

            dfs(new_row, new_col, visited)

    for row in range(rows):
        dfs(row, 0, pacific_reachable)
        dfs(row, cols - 1, atlantic_reachable)

    for col in range(cols):
        dfs(0, col, pacific_reachable)
        dfs(rows - 1, col, atlantic_reachable)

    result = []

    for row in range(rows):
        for col in range(cols):
            if (row, col) in pacific_reachable and (row, col) in atlantic_reachable:
                result.append([row, col])

    return result
