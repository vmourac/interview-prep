Valid Parentheses

Prompt: Dada uma string com parenteses, colchetes e chaves, decida se todos os fechamentos acontecem no tipo certo e na ordem certa.

Este exercicio e citado diretamente como problema relacionado na aula de stacks. A pagina abaixo transforma essa ideia em um roteiro curto para treino local.

Hints

  • Um fechamento so pode casar com a abertura mais recente ainda pendente.
  • Se a estrutura certa e LIFO, a pergunta sobre pilha deixa de ser opcional.
  • Falhe cedo quando surgir fechamento sem topo compativel.

Caminho ingenuo

Remover pares validos diretamente da string parece simples, mas revarre e reconstrui a entrada varias vezes. Em casos ruins, isso escala quadraticamente.

Caminho otimizado

Empilhe aberturas. Quando um fechamento aparece, confira se o topo contem a abertura correspondente. Se nao contiver, a resposta ja e falsa.

Complexidade

Tempo: O(n)

Espaco: O(n)

Cada simbolo entra na analise uma vez; as aberturas entram e saem da stack no maximo uma vez.

Solucao final

def isValid(s):
    pairs = {
        ')': '(',
        ']': '[',
        '}': '{',
    }
    stack = []

    for ch in s:
        if ch not in pairs:
            stack.append(ch)
            continue

        if not stack or stack[-1] != pairs[ch]:
            return False

        stack.pop()

    return len(stack) == 0
