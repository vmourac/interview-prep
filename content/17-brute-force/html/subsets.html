Subsets

Cada elemento abre duas portas: ignorar ou incluir. A árvore inteira materializa todos os 2^n subconjuntos.

include / skipbacktrackingO(n × 2^n)

Controles da simulação

Raciocínio atual

Saída domina: o custo exponencial é inevitável porque existem 2^n respostas.
ingênuamáscaras: O(n × 2^n)
otimizadaDFS clara: O(n × 2^n)

Solução ingênua

Iterar máscaras binárias funciona, mas comunica menos o padrão de decisão que reaparece em backtracking mais difícil.

Solução otimizada

Use DFS por índice: primeiro ignore nums[index], depois inclua, recurse e desfaça.

Copie o path quando chegar ao fim do array.

Big O

Tempo: O(n × 2^n).

Espaço: O(n × 2^n) para a saída e O(n) auxiliar.

Como falar em entrevista

  • Desenhe a decisão binária.
  • Explique por que não há duplicatas com elementos distintos.
  • Justifique o fator n pela cópia do caminho.

Código de referência

function subsets(nums) {
  const result = [];
  const path = [];

  function dfs(index) {
    if (index === nums.length) {
      result.push([...path]);
      return;
    }
    dfs(index + 1);
    path.push(nums[index]);
    dfs(index + 1);
    path.pop();
  }

  dfs(0);
  return result;
}
