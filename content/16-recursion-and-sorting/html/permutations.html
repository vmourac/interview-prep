Topico 16 - Roteiro de entrevista

Permutations

Problema classico para testar backtracking. A fonte do curso nao tem captions; esta pagina reconstroi o treino esperado em entrevistas a partir do tema da aula.

Prompt: Dado um array com valores distintos, retorne todas as permutacoes possiveis.

O que o entrevistador mede
  • Se voce identifica estado parcial e resposta final.
  • Se sabe desfazer escolhas ao voltar na recursao.
  • Se justifica por que o custo explode para n!.
Hint de ouro

Em cada profundidade, escolha um elemento ainda nao usado, avance e depois restaure o estado antes de tentar o proximo.

Complexidade

Tempo: O(n x n!)

Espaco auxiliar: O(n)

Se contar a saida, o armazenamento total sobe para O(n x n!).

Implementacao de referencia

def permute(nums):
    result = []
    path = []
    used = [False] * len(nums)

    def backtrack():
        if len(path) == len(nums):
            result.append(path[:])
            return

        for i in range(len(nums)):
            if used[i]:
                continue

            used[i] = True
            path.append(nums[i])

            backtrack()

            path.pop()
            used[i] = False

    backtrack()
    return result
