Valid Palindrome

Pagina editorial para revisar leitura de enunciado, limpeza de entrada e o padrao de dois ponteiros.

Prompt

Dada uma string, determine se ela e um palindromo considerando apenas caracteres alfanumericos e ignorando caixa.

Hints

Nao trate a string inteira como limpa desde o inicio. O enunciado exige filtrar caracteres e ignorar diferencas entre maiusculas e minusculas.

Caminho ingenuo

  • Filtre apenas os caracteres alfanumericos.
  • Converta tudo para minusculas.
  • Compare a string normalizada com sua reversa.

Caminho otimizado

  • Use um ponteiro no inicio e outro no fim.
  • Pule simbolos e espacos dos dois lados.
  • Compare apenas caracteres validos em lowercase.

Complexidade

Ingenua
Tempo O(n), espaco O(n)
Otimizada
Tempo O(n), espaco O(1)

O tempo continua linear porque cada caractere e lido no maximo algumas vezes. O ganho real esta em evitar a copia completa da string.

Solucao final

def is_palindrome(s):
    esquerda = 0
    direita = len(s) - 1

    while esquerda < direita:
        while esquerda < direita and not s[esquerda].isalnum():
            esquerda += 1
        while esquerda < direita and not s[direita].isalnum():
            direita -= 1

        if s[esquerda].lower() != s[direita].lower():
            return False

        esquerda += 1
        direita -= 1

    return True
