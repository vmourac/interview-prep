Binary Search

Use a ordenação para descartar metade do intervalo a cada comparação. O alvo, se existir, permanece sempre entre left e right.

Solução ingênua

Percorra o array do início ao fim comparando cada elemento com target. Funciona, mas ignora a pré-condição mais valiosa: a entrada já está ordenada.

  • Tempo: O(n), pois no pior caso visita todos os elementos.
  • Espaço: O(1).

Solução otimizada

Compare o meio do intervalo com o alvo. Se o meio é menor que o alvo, descarte a esquerda; se é maior, descarte a direita.

Invariante: se o alvo ainda puder existir, ele está no intervalo fechado [left, right].

Big O

Tempo otimizado: O(log n), porque cada passo reduz o intervalo pela metade.

Espaço otimizado: O(1), usando apenas ponteiros e o índice do meio.

Como falar em entrevista

  • Diga que a ordenação habilita busca binária.
  • Nomeie o invariante do intervalo.
  • Teste mentalmente alvo ausente, array vazio e um elemento.
  • Use left + Math.floor((right - left) / 2) como hábito seguro.

Código de exemplo

function search(nums, target) {
  let left = 0;
  let right = nums.length - 1;

  while (left <= right) {
    const mid = left + Math.floor((right - left) / 2);
    if (nums[mid] === target) return mid;
    if (nums[mid] < target) left = mid + 1;
    else right = mid - 1;
  }

  return -1;
}
