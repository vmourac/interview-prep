Caderno de entrevista

Two Sum

Guia sintetizado para treino de entrevista. A fonte do topico original e introdutoria e sem transcricao; por isso, este material organiza um problema canonico de arrays e hash map em formato de entrevistador.

Prompt

Dado um array de inteiros e um target, retorne os indices de dois elementos cuja soma seja exatamente igual ao target. Existe exatamente uma solucao valida.

Hints

Hint 1Pense no que falta para cada numero atingir o target.
Hint 2Se voce ja viu o complemento antes, a resposta esta pronta.
Hint 3Troque um segundo loop por memoria auxiliar.

Caminho ingenuo

Teste todos os pares com dois loops. E correto, mas compara pares demais.

  • Escolha um indice.
  • Compare com todos os indices seguintes.
  • Retorne quando a soma bater.

Caminho otimizado

Use um hash map de valor para indice. Para cada numero, procure o complemento antes de inserir o valor atual.

  • Calcule complemento = target - valor.
  • Se complemento ja existe no mapa, termine.
  • Senao, registre o valor atual.

Complexidade

Ingenua: tempo O(n^2)
Ingenua: espaco O(1)
Otimizada: tempo O(n)
Otimizada: espaco O(n)

A queda para O(n) acontece porque cada elemento e processado uma vez e cada consulta ao hash map custa O(1) em media. O espaco sobe para O(n) porque o mapa pode guardar quase todos os elementos.

Solucao final

def two_sum(nums, target):
    vistos = {}

    for indice, valor in enumerate(nums):
        complemento = target - valor
        if complemento in vistos:
            return [vistos[complemento], indice]
        vistos[valor] = indice

    return []
