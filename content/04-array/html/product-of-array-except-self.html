Topico 04 - Laboratorio editorial

Product of Array Except Self

Material local e autocontido para revisar um padrao muito cobrado em entrevista: decompor a resposta de cada indice em produto da esquerda e produto da direita. Como a aula-fonte nao tem transcript, esta pagina e uma sintese guiada do topico.

Prompt: Retorne um array em que cada posicao contenha o produto de todos os outros elementos, sem usar divisao.

Hints
  • O que falta para um indice pode ser dividido em parte esquerda e parte direita.
  • Pense em duas passadas, nao em um loop dentro de outro.
  • O proprio array de resposta pode armazenar trabalho parcial.
Caminho ingenuo

Multiplicar todos os outros elementos para cada indice resolve o enunciado, mas repete quase todo o trabalho `n` vezes.

Caminho otimizado

Primeiro registre o produto acumulado da esquerda. Depois multiplique por um acumulado da direita em uma passada inversa.

Complexidade

Tempo: O(n)

Espaco: O(1) auxiliar

O custo fica linear porque cada elemento participa de duas atualizacoes constantes e nao ha arrays auxiliares extras alem da saida.

Solucao final

def productExceptSelf(nums):
    n = len(nums)
    answer = [1] * n

    prefix = 1
    for i in range(n):
        answer[i] = prefix
        prefix *= nums[i]

    suffix = 1
    for i in range(n - 1, -1, -1):
        answer[i] *= suffix
        suffix *= nums[i]

    return answer
