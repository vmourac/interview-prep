Reverse Linked List

O objetivo não é decorar o código. É enxergar a lista como um conjunto de arestas que podem ser invertidas uma por vez sem perder acesso ao restante da estrutura.

padrão: prev / current / next otimizado: O(n) tempo otimizado: O(1) espaço

Controles da simulação

Raciocínio atual

Armadilha clássica: se você fizer current.next = prev antes de salvar nextNode, perde o restante da lista.
ingênuaO(n) tempo · O(n) espaço
otimizadaO(n) tempo · O(1) espaço

Solução ingênua

Uma forma direta é copiar os valores para um array e reconstruir a lista do fim para o começo. Isso confirma o entendimento do resultado esperado, mas não treina manipulação de referências.

  • Tempo: percorre os nós e reconstrói, então O(n).
  • Espaço: armazena todos os valores, então O(n).

Solução otimizada

A cada iteração, salve o próximo nó, aponte o nó atual para a parte já revertida e avance os ponteiros. A lista é revertida localmente, aresta por aresta.

Invariante: antes de cada iteração, prev aponta para a parte já revertida e current aponta para o primeiro nó ainda não processado.

Big O

Tempo otimizado: O(n), porque cada nó é visitado exatamente uma vez.

Espaço otimizado: O(1), porque apenas três ponteiros auxiliares são mantidos independentemente do tamanho da lista.

Como falar em entrevista

  • Declare os papéis de prev, current e nextNode.
  • Explique por que salvar nextNode vem antes de sobrescrever current.next.
  • Use um exemplo pequeno, como 1 → 2 → 3, para provar o invariante.
  • Finalize com o novo head: quando current vira null, prev está no último nó processado.

Código de referência

function reverseList(head) {
  let prev = null;
  let current = head;

  while (current !== null) {
    const nextNode = current.next;
    current.next = prev;
    prev = current;
    current = nextNode;
  }

  return prev;
}
type ListNode = {
  value: number;
  next: ListNode | null;
};

function reverseList(head: ListNode | null): ListNode | null {
  let prev: ListNode | null = null;
  let current: ListNode | null = head;

  while (current !== null) {
    const nextNode: ListNode | null = current.next;
    current.next = prev;
    prev = current;
    current = nextNode;
  }

  return prev;
}
