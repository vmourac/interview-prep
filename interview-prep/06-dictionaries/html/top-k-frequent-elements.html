Solução ingênua

Conte frequências e ordene todos os m valores distintos por frequência decrescente. Simples, mas paga sort global e usa O(m) espaço para o mapa e a lista ordenada.

Solução otimizada

Coloque cada valor no bucket correspondente à sua frequência. Depois percorra os buckets do maior para o menor até coletar k valores.

Ideia-chave: mantenha um estado pequeno e reutilizável em vez de reprocessar o histórico.

Big O

A abordagem ingênua custa O(n + m log m) tempo e O(m) espaço. Na otimizada, contar, distribuir e percorrer buckets são operações lineares limitadas por n, com O(n) espaço.

Como falar em entrevista

Defina m como número de valores distintos e explique por que buckets exploram o limite natural de frequência: 1..n.

Código de referência

function topKFrequent(nums, k) {
  const freq = new Map();
  for (const value of nums) {
    freq.set(value, (freq.get(value) || 0) + 1);
  }

  const buckets = Array.from({ length: nums.length + 1 }, () => []);
  for (const [value, count] of freq) {
    buckets[count].push(value);
  }

  const result = [];
  for (let count = buckets.length - 1; count > 0; count--) {
    for (const value of buckets[count]) {
      result.push(value);
      if (result.length === k) return result;
    }
  }

  return result;
}
