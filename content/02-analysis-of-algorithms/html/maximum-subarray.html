Maximum Subarray

Em cada posição, decida se vale a pena continuar o subarray anterior ou recomeçar no valor atual. Prefixo negativo nunca ajuda o futuro.

Raciocínio atual

Solução ingênua

Escolha todo início possível e expanda o fim acumulando a soma, atualizando a melhor resposta global.

  • Tempo: O(n²), há muitos intervalos contíguos.
  • Espaço: O(1).

Solução otimizada

Kadane mantém a melhor soma que termina no índice atual (current) e a melhor soma global (best).

Decisão: current = max(value, current + value). Se o passado atrapalha, recomece.

Big O

Tempo otimizado: O(n), um único percurso.

Espaço otimizado: O(1), só acumuladores.

Como falar em entrevista

  • Enfatize que o subarray precisa ser contíguo.
  • Diga que prefixos negativos são descartáveis.
  • Inicialize com nums[0] para lidar com arrays todos negativos.
  • Atualize best depois de decidir o novo current.

Código de exemplo

function maxSubArray(nums) {
  let current = nums[0];
  let best = nums[0];

  for (let i = 1; i < nums.length; i += 1) {
    current = Math.max(nums[i], current + nums[i]);
    best = Math.max(best, current);
  }

  return best;
}
