Terrain Radar

Path With Minimum Effort

A leitura correta e minimax: o custo do caminho nao e a soma dos saltos, mas o pior salto que voce foi obrigado a aceitar ate o destino.

1 2 2 3 3 8 2 1
Signal: minimize o maior salto
State: (effort, row, col)
Relax: max(current, edge)
Tempo: O(R * C * log(R * C))

Prompt

Encontre um caminho do canto superior esquerdo ao inferior direito cujo pior degrau entre celulas adjacentes seja o menor possivel.

Hints

  1. Pense em custo do caminho, nao de uma unica aresta.
  2. O custo novo e o maximo entre esforco atual e salto local.
  3. A heap deve ordenar pelo menor pior salto conhecido.

Naive Path

Backtracking por todos os caminhos da grade tenta combinacoes demais e recalcula varias vezes o mesmo estado com esforcos piores.

Optimized Path

Use Dijkstra minimax. A melhor fronteira e a celula cujo maior salto acumulado ate agora e o menor entre todos os candidatos vivos.

Interview Note

Este problema diferencia quem memoriza Dijkstra de quem entende o padrao. A soma foi trocada por uma funcao monotona de maximo.

Final Solution

import heapq


def minimumEffortPath(heights: list[list[int]]) -> int:
    rows = len(heights)
    cols = len(heights[0])
    inf = float("inf")

    effort = [[inf] * cols for _ in range(rows)]
    effort[0][0] = 0
    heap = [(0, 0, 0)]
    directions = ((1, 0), (-1, 0), (0, 1), (0, -1))

    while heap:
        current_effort, row, col = heapq.heappop(heap)

        if row == rows - 1 and col == cols - 1:
            return current_effort

        if current_effort > effort[row][col]:
            continue

        for delta_row, delta_col in directions:
            next_row = row + delta_row
            next_col = col + delta_col

            if not (0 <= next_row < rows and 0 <= next_col < cols):
                continue

            edge_cost = abs(heights[row][col] - heights[next_row][next_col])
            next_effort = max(current_effort, edge_cost)

            if next_effort < effort[next_row][next_col]:
                effort[next_row][next_col] = next_effort
                heapq.heappush(heap, (next_effort, next_row, next_col))

    return 0
