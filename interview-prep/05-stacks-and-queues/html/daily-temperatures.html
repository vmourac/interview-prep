Daily Temperatures

Use uma pilha decrescente de índices para resolver dias pendentes quando aparece uma temperatura maior.

Controles da simulação

Raciocínio atual

Atenção: Temperaturas iguais não resolvem o dia anterior; a comparação precisa ser estritamente maior.
ingênuaO(n²) tempo · O(1) extra
otimizadaO(n) tempo · O(n) espaço

Solução ingênua

Para cada dia, olhe todos os dias futuros até encontrar o primeiro mais quente. Em sequência decrescente, isso vira quadrático.

Solução otimizada

Mantenha uma stack de dias ainda sem resposta. Quando a temperatura atual supera o topo, desempilhe e preencha a espera.

Ideia-chave: mantenha um estado pequeno e reutilizável em vez de reprocessar o histórico.

Big O

Cada índice entra e sai da stack no máximo uma vez; por isso o while total é linear ao longo do algoritmo.

Como falar em entrevista

Chame atenção para análise amortizada: o while pode rodar várias vezes em uma iteração, mas consome índices que nunca voltam.

Código de referência

function dailyTemperatures(temperatures) {
  const answer = new Array(temperatures.length).fill(0);
  const stack = [];

  for (let i = 0; i < temperatures.length; i++) {
    while (stack.length && temperatures[i] > temperatures[stack[stack.length - 1]]) {
      const previous = stack.pop();
      answer[previous] = i - previous;
    }
    stack.push(i);
  }

  return answer;
}
function dailyTemperatures(temperatures: number[]): number[] {
  const answer = new Array<number>(temperatures.length).fill(0);
  const stack: number[] = [];

  for (let i = 0; i < temperatures.length; i++) {
    while (stack.length > 0 && temperatures[i] > temperatures[stack[stack.length - 1]]) {
      const previous = stack.pop()!;
      answer[previous] = i - previous;
    }
    stack.push(i);
  }

  return answer;
}
