Topico 16 - Divide and Conquer

Sort an Array

Problema tipico para separar ordenacoes quadrativas de algoritmos O(n log n). Como a fonte do curso nao tem captions, esta pagina organiza a explicacao em modo entrevista.

Prompt: Dado um array de inteiros, retorne-o em ordem crescente.

Ponto de comparacao

Bubble, selection e insertion sort ajudam como baseline, mas a resposta forte aqui costuma ser merge sort quando o objetivo e previsibilidade em O(n log n).

Intuicao
  • Divida em metades menores ate casos triviais.
  • Ordene cada metade recursivamente.
  • Mescle em tempo linear no tamanho combinado.
Complexidade

Tempo: O(n log n)

Espaco: O(n)

Ha O(log n) niveis e O(n) de trabalho agregado por nivel.

Implementacao de referencia

def sortArray(nums):
    if len(nums) <= 1:
        return nums

    mid = len(nums) // 2
    left = sortArray(nums[:mid])
    right = sortArray(nums[mid:])

    merged = []
    i = 0
    j = 0

    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            merged.append(left[i])
            i += 1
        else:
            merged.append(right[j])
            j += 1

    merged.extend(left[i:])
    merged.extend(right[j:])
    return merged
