Alien Dictionary

Este problema mede se voce sabe extrair restricoes minimas do input antes de rodar topological sort. O grafo nasce entre letras, nao entre palavras.

wrt
wrf
er
ett
rftt
Signal: inferir arestas de precedencia
Tempo: O(S + C + E)
Espaco: O(C + E)
Edge case: prefixo invalido

Prompt

Dado um dicionario ja ordenado em uma lingua desconhecida, devolva uma ordem valida de letras ou string vazia.

Hints

  1. Compare apenas palavras adjacentes.
  2. A primeira diferenca produz a unica aresta segura.
  3. Se palavra maior vier antes do proprio prefixo, o input e invalido.

Naive Path

Tentar adivinhar uma ordem global de letras por tentativa e erro confunde a parte dificil do problema: extrair as restricoes corretas.

Optimized Path

Construa o grafo letra a letra a partir da primeira divergencia entre palavras adjacentes e depois rode Kahn.

Interview Note

Se voce nao mencionar o caso de prefixo invalido, a solucao parece incompleta mesmo que o topological sort esteja correto.

Final Solution

from collections import deque


def alienOrder(words: list[str]) -> str:
    graph = {char: set() for word in words for char in word}
    indegree = {char: 0 for char in graph}

    for index in range(len(words) - 1):
        first = words[index]
        second = words[index + 1]

        if len(first) > len(second) and first.startswith(second):
            return ""

        for left, right in zip(first, second):
            if left != right:
                if right not in graph[left]:
                    graph[left].add(right)
                    indegree[right] += 1
                break

    queue = deque(char for char in indegree if indegree[char] == 0)
    order = []

    while queue:
        char = queue.popleft()
        order.append(char)

        for neighbor in graph[char]:
            indegree[neighbor] -= 1
            if indegree[neighbor] == 0:
                queue.append(neighbor)

    return "".join(order) if len(order) == len(graph) else ""
