Tempo: O(E * alpha(V))

Ferramenta: Union-Find

Prompt

Dado `n` e uma lista de arestas nao dirigidas, decida se o grafo forma uma arvore valida.

Hints

  1. Uma arvore com `n` vertices precisa de `n - 1` arestas.
  2. Se uma uniao falha, apareceu ciclo.
  3. Nao trate conectividade e ciclo como ideias separadas no discurso final.

Naive Path

Rodar DFS repetida para revalidar conectividade ou ciclo a cada aresta funciona, mas repete exploracao do grafo varias vezes.

Optimized Path

Primeiro valide o numero de arestas. Depois use Union-Find para rejeitar qualquer aresta que conecte vertices ja na mesma componente.

Interview Note

Uma fala madura e: se tenho `n - 1` arestas e nenhum ciclo, o grafo necessariamente fica conexo e portanto e uma arvore.

Final Solution

def validTree(n: int, edges: list[list[int]]) -> bool:
    if len(edges) != n - 1:
        return False

    parent = list(range(n))
    rank = [0] * n

    def find(node: int) -> int:
        if parent[node] != node:
            parent[node] = find(parent[node])
        return parent[node]

    def union(a: int, b: int) -> bool:
        root_a = find(a)
        root_b = find(b)

        if root_a == root_b:
            return False

        if rank[root_a] < rank[root_b]:
            parent[root_a] = root_b
        elif rank[root_a] > rank[root_b]:
            parent[root_b] = root_a
        else:
            parent[root_b] = root_a
            rank[root_a] += 1

        return True

    for a, b in edges:
        if not union(a, b):
            return False

    return True
