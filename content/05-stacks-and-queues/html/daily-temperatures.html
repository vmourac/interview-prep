Topico 05 · Monotonic stack

Daily Temperatures

Prompt: Para cada dia, informe quantos dias faltam ate aparecer uma temperatura estritamente maior.

Fonte indireta: o curso menciona monotonic deque e Sliding Window Maximum; esta pagina usa Daily Temperatures como treino equivalente de monotonic structure.

Hints

  • Voce nao quer comparar cada dia com todos os seguintes.
  • A estrutura certa precisa lembrar apenas dias ainda nao resolvidos.
  • Guarde indices, nao apenas valores, porque a resposta depende da distancia.

Caminho ingenuo

Para cada dia, varrer todos os proximos dias funciona, mas repete comparacoes demais. Em sequencias decrescentes, o custo explode para O(n^2).

Caminho otimizado

Mantenha uma stack monotonic decrescente de indices. Quando surge uma temperatura maior, ela resolve em lote os dias do topo que estavam esperando por esse momento.

Complexidade

Tempo: O(n)

Espaco: O(n)

Cada indice entra e sai da stack no maximo uma vez. Esse e o argumento amortizado que protege o algoritmo contra leitura superficial de "for + while".

Solucao final

def dailyTemperatures(temperatures):
    answer = [0] * len(temperatures)
    stack = []

    for index, current in enumerate(temperatures):
        while stack and current > temperatures[stack[-1]]:
            prev_index = stack.pop()
            answer[prev_index] = index - prev_index

        stack.append(index)

    return answer
