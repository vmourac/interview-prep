Signal: fewest transformations
Graph: implicito e nao ponderado
Tempo: O(N * L^2)
Hint: patterns com `*`

Prompt

Transforme `beginWord` em `endWord` mudando um caractere por vez. Toda palavra intermediaria precisa existir em `wordList`. Retorne o comprimento da menor sequencia; se nao houver, retorne `0`.

Hints

  1. O grafo pode ser construido implicitamente.
  2. Comparar toda palavra com toda palavra funciona, mas custa caro.
  3. Padroes como `h*t` e `*ot` agrupam vizinhos de forma eficiente.

Naive Path

Construir todas as arestas comparando todos os pares de palavras leva a um preprocessamento quadraticamente pesado antes mesmo da busca comecar.

Optimized Path

Indexe palavras por padroes intermediarios. A BFS percorre palavras em ordem de numero de transformacoes e o primeiro encontro com `endWord` fecha a menor sequencia.

Interview Note

O diferencial aqui e verbalizar que o problema e shortest path em grafo nao ponderado, mas o grafo e implicito. A BFS entra junto com uma estrategia de geracao eficiente de vizinhos.

Final Solution

from collections import defaultdict, deque


def ladderLength(beginWord: str, endWord: str, wordList: list[str]) -> int:
    if endWord not in wordList:
        return 0

    pattern_map = defaultdict(list)
    word_length = len(beginWord)

    for word in wordList:
        for index in range(word_length):
            pattern = word[:index] + "*" + word[index + 1:]
            pattern_map[pattern].append(word)

    queue = deque([(beginWord, 1)])
    visited = {beginWord}

    while queue:
        word, steps = queue.popleft()

        if word == endWord:
            return steps

        for index in range(word_length):
            pattern = word[:index] + "*" + word[index + 1:]

            for neighbor in pattern_map[pattern]:
                if neighbor in visited:
                    continue

                visited.add(neighbor)
                queue.append((neighbor, steps + 1))

            pattern_map[pattern] = []

    return 0
