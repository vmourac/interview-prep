Subarray Sum Equals K

Este problema testa se voce enxerga o dicionario como historico de estados, nao apenas como tabela de membership. A pergunta decisiva e: quantas vezes ja apareceu um prefixo que complete a soma desejada?

Prompt

Dado um array de inteiros e um inteiro `k`, retorne quantos subarrays contiguos somam exatamente `k`.

Hints

  1. Se eu conheco a soma acumulada ate agora, que informacao do passado me diz se existe um intervalo valido terminando aqui?
  2. Como representar todos os prefixos anteriores sem voltar no array?
  3. Por que faz sentido iniciar o mapa com `0: 1`?

Naive Path

Fixe um inicio e expanda o fim calculando a soma de cada subarray. Mesmo aproveitando a soma incremental, ainda existem O(n^2) intervalos a examinar.

Optimized Path

Mantenha o prefixo atual e um mapa de quantas vezes cada prefixo anterior apareceu. Sempre que `prefix - k` ja existir, cada ocorrencia dele define um subarray valido terminando na posicao atual.

Complexity

  • Tempo: O(n), por um unico percurso com operacoes medias O(1) no mapa.
  • Espaco: O(n), porque prefixos distintos podem crescer linearmente.
  • Entrevista: explique que o mapa guarda contagem de prefixos, nao apenas presenca.

Final Solution

def subarraySum(nums: list[int], k: int) -> int:
    prefix = 0
    result = 0
    count = {0: 1}

    for value in nums:
        prefix += value
        result += count.get(prefix - k, 0)
        count[prefix] = count.get(prefix, 0) + 1

    return result
