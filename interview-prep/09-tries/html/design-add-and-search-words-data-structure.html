Wildcard Search

Design Add and Search Words
Data Structure

O salto deste problema e perceber que trie sozinha resolve o caminho literal, mas o caractere `.` exige DFS e branching controlado.

synthesized study guide
Add: O(L)
Search: pior caso O(B^L)
Stored structure: O(S)
Call stack: O(L)

Prompt

Implemente `addWord` e `search`, onde `.` representa qualquer letra unica durante a busca.

Hints

  1. Sem `.` a busca e um caminho unico.
  2. Com `.` voce precisa tentar todos os filhos do no atual.
  3. Se um ramo falhar cedo, corte imediatamente.

Naive Path

Guardar todas as palavras e comparar cada consulta contra todas as strings de mesmo tamanho. O wildcard apenas muda a regra de comparacao; ele nao reduz o volume de trabalho.

Optimized Path

Use trie para armazenar palavras e uma DFS recursiva para consultas com `.`. Cada wildcard abre varios ramos, mas ramos impossiveis morrem cedo pelo proprio formato da trie.

Interview Note

Uma resposta forte separa comportamento pratico e pior caso teorico: pruning costuma ajudar muito, mas o pior caso segue exponencial quando a consulta e dominada por `.`.

Final Solution

class TrieNode:
    def __init__(self):
        self.children = {}
        self.is_end = False


class WordDictionary:
    def __init__(self):
        self.root = TrieNode()

    def addWord(self, word: str) -> None:
        node = self.root

        for char in word:
            if char not in node.children:
                node.children[char] = TrieNode()
            node = node.children[char]

        node.is_end = True

    def search(self, word: str) -> bool:
        def dfs(index: int, node: TrieNode) -> bool:
            if index == len(word):
                return node.is_end

            char = word[index]

            if char == ".":
                for child in node.children.values():
                    if dfs(index + 1, child):
                        return True
                return False

            if char not in node.children:
                return False

            return dfs(index + 1, node.children[char])

        return dfs(0, self.root)
