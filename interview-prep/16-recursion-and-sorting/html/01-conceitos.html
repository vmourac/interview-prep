Raciocínio atual

Regra mental: se você não consegue nomear o menor caso e o retorno de cada chamada, a recursão ainda está nebulosa.
ingênuaO(n²) tempo · O(1) espaço
otimizadaO(n log n) tempo · O(n) espaço

Solução ingênua

Para ordenar, um baseline como insertion sort compara e move elementos repetidamente. É simples, mas o pior caso cresce como O(n²).

Solução otimizada

Merge sort divide o array em metades, resolve recursivamente cada metade e combina duas metades já ordenadas em tempo linear.

Promessa: mergeSort(nums) devolve uma nova versão ordenada de nums.

Big O

Tempo: O(n log n): há O(log n) níveis e cada nível soma O(n) de merge.

Espaço: O(n) para buffers de intercalação, mais stack O(log n).

Como falar em entrevista

  • Declare a promessa da função recursiva.
  • Mostre o caso base: array de tamanho 0 ou 1.
  • Explique a combinação: merge linear.
  • Justifique stack e buffer separadamente.

Código de referência

function mergeSort(nums) {
  if (nums.length <= 1) return nums.slice();

  const mid = Math.floor(nums.length / 2);
  const left = mergeSort(nums.slice(0, mid));
  const right = mergeSort(nums.slice(mid));
  return merge(left, right);
}

function merge(left, right) {
  const out = [];
  let i = 0;
  let j = 0;

  while (i < left.length && j < right.length) {
    if (left[i] <= right[j]) out.push(left[i++]);
    else out.push(right[j++]);
  }

  return out.concat(left.slice(i), right.slice(j));
}
