Prompt

Dadas dependencias entre cursos, diga se existe uma forma de concluir todos eles sem quebrar prerequisitos.

Hints

  1. Pergunte primeiro se o grafo e aciclico.
  2. `visited` sozinho nao basta em grafo dirigido.
  3. Reencontrar um no ainda aberto na pilha recursiva denuncia o ciclo.

Naive Path

Simular varias agendas de cursos e tentar ver se alguma funciona explode em combinacoes e ignora a estrutura certa do problema.

Optimized Path

Use DFS com tres estados. Se um vizinho reaparecer em estado visitando, existe back edge e portanto ciclo.

Interview Note

O entrevistador quer ouvir que DAG implica ordenacao topologica possivel. A DFS serve para responder exatamente a parte do ciclo.

Final Solution

def canFinish(numCourses: int, prerequisites: list[list[int]]) -> bool:
    graph = [[] for _ in range(numCourses)]

    for course, prereq in prerequisites:
        graph[prereq].append(course)

    state = [0] * numCourses

    def dfs(course: int) -> bool:
        if state[course] == 1:
            return False
        if state[course] == 2:
            return True

        state[course] = 1

        for neighbor in graph[course]:
            if not dfs(neighbor):
                return False

        state[course] = 2
        return True

    for course in range(numCourses):
        if state[course] == 0 and not dfs(course):
            return False

    return True
