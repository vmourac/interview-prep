Maximum Subarray

Aqui o entrevistador quer ver a virada mental: sair da enumeracao de todos os intervalos e perceber que um prefixo ruim pode ser descartado imediatamente. E um teste de clareza sobre por que uma decisao local ainda preserva a melhor resposta global.

Prompt

Dado um array de inteiros, encontre a maior soma possivel entre todos os subarrays contiguos nao vazios.

Hints

  1. Se a soma acumulada anterior ficou negativa, ela ajuda ou atrapalha?
  2. Voce precisa guardar todos os subarrays ou apenas o melhor que termina na posicao atual?
  3. Como atualizar a melhor resposta global durante um unico percurso?

Naive Path

Escolha cada indice inicial e expanda o fim do intervalo. Mesmo reaproveitando a soma conforme expande, ainda existem O(n^2) subarrays contiguos a considerar.

Optimized Path

Para cada valor, escolha entre recomecar nele ou estender o subarray anterior. Se o passado tornou a soma pior do que recomecar, descarte-o. Esse e o nucleo do algoritmo de Kadane.

Complexity

  • Tempo: O(n), porque ha um unico percurso e custo constante por elemento.
  • Espaco: O(1), porque so duas variaveis acompanham estado local e melhor resposta.
  • Entrevista: explique por que um prefixo negativo nunca ajuda um subarray futuro.

Final Solution

def maxSubArray(nums: list[int]) -> int:
    best = nums[0]
    current = nums[0]

    for value in nums[1:]:
        current = max(value, current + value)
        best = max(best, current)

    return best
