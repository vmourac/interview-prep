Network Delay Time

O alvo da explicacao e mostrar shortest path classico em grafo dirigido ponderado e depois traduzir a resposta final como o maior tempo entre todas as menores distancias.

K 2 ms 5 ms all reached?
Signal: menor caminho com pesos nao negativos
Heap: (dist, node)
Tempo: O((V + E) log V)
Final read: maximo das menores distancias

Prompt

Um sinal parte de um no inicial e percorre arestas com atrasos diferentes. Descubra quanto tempo o ultimo no leva para receber o sinal, ou diga que alguem nunca recebe.

Hints

  1. Monte lista de adjacencia dirigida.
  2. Use Dijkstra a partir do no inicial.
  3. Ao final, pegue o maximo entre as menores distancias.

Naive Path

Tentar enumerar rotas ou repetir busca sem consolidar a melhor distancia por no faz o problema explodir em caminhos redundantes.

Optimized Path

Use min-heap para sempre expandir o menor atraso acumulado. Cada relaxamento tenta reduzir a melhor distancia conhecida do vizinho.

Interview Note

O no mais distante entre os alcancaveis define o tempo total de propagacao. Se algum continuar em infinito, a resposta vira -1.

Final Solution

import heapq


def networkDelayTime(times: list[list[int]], n: int, k: int) -> int:
    graph = [[] for _ in range(n + 1)]

    for source, target, weight in times:
        graph[source].append((target, weight))

    inf = float("inf")
    dist = [inf] * (n + 1)
    dist[k] = 0
    heap = [(0, k)]

    while heap:
        current_dist, node = heapq.heappop(heap)

        if current_dist > dist[node]:
            continue

        for neighbor, weight in graph[node]:
            new_dist = current_dist + weight
            if new_dist < dist[neighbor]:
                dist[neighbor] = new_dist
                heapq.heappush(heap, (new_dist, neighbor))

    answer = max(dist[1:])
    return -1 if answer == inf else answer
