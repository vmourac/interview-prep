Raciocínio atual

Foco de entrevista: Quando encontrar ponto, explore filhos; quando encontrar literal, siga um único ramo.
ingênuaadd O(1) amort. · search O(NL) · espaço O(S)
otimizadaadd O(L) · search pior O(B^L) · espaço O(S)

Solução ingênua

Guarde todas as palavras e compare a consulta contra cada uma, tratando ponto como qualquer caractere.

  • Busca pode testar N palavras.
  • Cada comparação pode consumir L caracteres.

Solução otimizada

Use trie para adicionar palavras. Na busca, literais seguem uma aresta; ponto tenta todos os filhos com DFS.

Invariante: no índice i, a chamada DFS representa todos os caminhos que casaram o prefixo da consulta até i.

Big O

addWord custa O(L). search sem wildcard é O(L); com muitos pontos pode ramificar até O(B^L).

Como falar em entrevista

  • Mostre pruning: ramo sem próximo literal morre cedo.
  • Explique que ponto casa exatamente um caractere.
  • Destaque que terminal importa no fim da palavra.

Código de referência

class TrieNode { constructor() { this.children = new Map(); this.isEnd = false; } }
class WordDictionary {
  constructor() { this.root = new TrieNode(); }
  addWord(word) { let node = this.root; for (const ch of word) { if (!node.children.has(ch)) node.children.set(ch, new TrieNode()); node = node.children.get(ch); } node.isEnd = true; }
  search(word) {
    const dfs = (i, node) => {
      if (i === word.length) return node.isEnd;
      const ch = word[i];
      if (ch === '.') return [...node.children.values()].some(child => dfs(i + 1, child));
      return node.children.has(ch) && dfs(i + 1, node.children.get(ch));
    };
    return dfs(0, this.root);
  }
}
class TrieNode { children = new Map<string, TrieNode>(); isEnd = false; }
class WordDictionary {
  private root = new TrieNode();
  addWord(word: string): void { let node = this.root; for (const ch of word) { if (!node.children.has(ch)) node.children.set(ch, new TrieNode()); node = node.children.get(ch)!; } node.isEnd = true; }
  search(word: string): boolean {
    const dfs = (i: number, node: TrieNode): boolean => {
      if (i === word.length) return node.isEnd;
      const ch = word[i];
      if (ch === '.') return [...node.children.values()].some((child) => dfs(i + 1, child));
      const next = node.children.get(ch);
      return next !== undefined && dfs(i + 1, next);
    };
    return dfs(0, this.root);
  }
}
