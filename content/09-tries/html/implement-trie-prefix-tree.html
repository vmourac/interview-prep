Implement Trie
(Prefix Tree)

Este prompt mede se voce sabe transformar palavras em caminhos, compartilhar prefixos e distinguir prefixo valido de palavra completa.

Nota de fonte: este material e sintetizado a partir do topico do curso, porque a aula original nao tem captions nem transcript disponiveis.

Insert: O(L)
Search: O(L)
StartsWith: O(L)
Space: O(S)

Prompt

Implemente `insert`, `search` e `startsWith` para uma estrutura de trie com letras minusculas.

Hints

  1. Qual e a diferenca entre existir um caminho e existir uma palavra completa?
  2. Onde voce guarda a informacao de terminal?
  3. Por que cada operacao consome apenas um caractere por passo?

Naive Path

Guardar palavras em lista e responder `search` e `startsWith` por varredura direta. Funciona, mas nao reaproveita prefixos e custa `O(N * L)` por consulta.

Optimized Path

Represente cada palavra como um caminho na trie. Prefixos compartilhados reutilizam os mesmos nos, e `startsWith` vira apenas a verificacao da existencia do caminho.

Interview Note

Uma resposta forte deixa claro que `search` exige `is_end = True`, enquanto `startsWith` exige apenas que o caminho exista.

Final Solution

class TrieNode:
    def __init__(self):
        self.children = {}
        self.is_end = False


class Trie:
    def __init__(self):
        self.root = TrieNode()

    def insert(self, word: str) -> None:
        node = self.root

        for char in word:
            if char not in node.children:
                node.children[char] = TrieNode()
            node = node.children[char]

        node.is_end = True

    def search(self, word: str) -> bool:
        node = self.root

        for char in word:
            if char not in node.children:
                return False
            node = node.children[char]

        return node.is_end

    def startsWith(self, prefix: str) -> bool:
        node = self.root

        for char in prefix:
            if char not in node.children:
                return False
            node = node.children[char]

        return True
