Rotting Oranges

Várias laranjas podres começam ao mesmo tempo; BFS multi-source mede minutos por ondas de contágio.

Raciocínio atual

Foco de entrevista: Coloque todas as podres iniciais na fila antes de começar; cada camada da BFS representa um minuto.
ingênuaO((R · C)²) tempo
otimizadaO(R · C) tempo · O(R · C) espaço

Solução ingênua

Simular minuto por minuto varrendo a grade inteira para achar novos apodrecimentos funciona, mas repete leituras globais até não mudar mais nada.

Solução otimizada

Enfileire todas as laranjas podres iniciais com minuto 0. Cada laranja fresca vizinha vira podre uma vez e entra na fila com minuto + 1.

Big O

Tempo: O(R · C), cada célula entra na fila no máximo uma vez.

Espaço: O(R · C) no pior caso para a fila.

Explicação de entrevista

Na entrevista, comece modelando o grafo e nomeie a estrutura que evita repetição. Depois caminhe por um exemplo pequeno, declare o invariante e só então escreva o código. Feche justificando tempo e espaço em termos de vértices/arestas ou células.

Código de exemplo

function orangesRotting(grid) {
  const rows = grid.length, cols = grid[0].length;
  const queue = [];
  let fresh = 0;
  for (let r = 0; r < rows; r++) for (let c = 0; c < cols; c++) {
    if (grid[r][c] === 2) queue.push([r, c, 0]);
    if (grid[r][c] === 1) fresh++;
  }
  let minutes = 0;
  for (let head = 0; head < queue.length; head++) {
    const [r, c, minute] = queue[head];
    minutes = Math.max(minutes, minute);
    for (const [dr, dc] of [[1,0],[-1,0],[0,1],[0,-1]]) {
      const nr = r + dr, nc = c + dc;
      if (nr < 0 || nr >= rows || nc < 0 || nc >= cols || grid[nr][nc] !== 1) continue;
      grid[nr][nc] = 2; fresh--; queue.push([nr, nc, minute + 1]);
    }
  }
  return fresh === 0 ? minutes : -1;
}
