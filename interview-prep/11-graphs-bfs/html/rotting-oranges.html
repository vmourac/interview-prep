Signal: spread by minutes
Technique: multi-source BFS
Tempo: O(R * C)
Espaco: O(R * C)

Prompt

Dado um grid com laranjas frescas e podres, calcule quantos minutos sao necessarios para toda laranja fresca adjacente a uma podre tambem apodrecer. Se alguma nunca puder apodrecer, retorne `-1`.

Hints

  1. Todas as laranjas podres iniciais devem entrar juntas na fila.
  2. O contador de minutos sobe por camada processada, nao por celula.
  3. Um contador de frescas simplifica o caso impossivel.

Naive Path

Escanear o grid inteiro a cada minuto funciona, mas repete trabalho demais porque revisita celulas que ja estao estabilizadas.

Optimized Path

Inicialize a BFS com todas as celulas podres. Quando uma fresca entra na fila, ela ja fica marcada como podre. Isso preserva a simultaneidade e evita duplicatas.

Interview Note

O erro mais comum e rodar varias BFS separadas. Isso quebra a intuicao do "tempo minimo global" porque as fontes deveriam competir em paralelo, nao em sequencia.

Final Solution

from collections import deque


def orangesRotting(grid: list[list[int]]) -> int:
    rows = len(grid)
    cols = len(grid[0]) if grid else 0
    queue = deque()
    fresh = 0

    for row in range(rows):
        for col in range(cols):
            if grid[row][col] == 2:
                queue.append((row, col))
            elif grid[row][col] == 1:
                fresh += 1

    if fresh == 0:
        return 0

    minutes = 0

    while queue and fresh > 0:
        for _ in range(len(queue)):
            row, col = queue.popleft()

            for delta_row, delta_col in ((1, 0), (-1, 0), (0, 1), (0, -1)):
                new_row = row + delta_row
                new_col = col + delta_col

                if not (0 <= new_row < rows and 0 <= new_col < cols):
                    continue

                if grid[new_row][new_col] != 1:
                    continue

                grid[new_row][new_col] = 2
                fresh -= 1
                queue.append((new_row, new_col))

        minutes += 1

    return minutes if fresh == 0 else -1
